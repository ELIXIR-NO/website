## Introduction

[JASPAR](https://jaspar.elixir.no/) is an open-access database of manually curated, non-redundant **transcription factor (TF) binding profiles** for eukaryotes. Supported by ELIXIR Norway, the database contains profiles derived from published collections of experimentally defined TF binding sites. Its key differentiators from similar resources (such as TRANSFAC) are **open data access**, **non-redundancy**, and **quality**.

## Key Features

- **Curated binding profiles** — manually verified, non-redundant TF binding site models backed by experimental evidence
- **Multi-level search** — search by organism, gene, or protein, or browse across 6 taxonomic groups
- **Programmatic access** — RESTful API and **pyJASPAR** Python interface for automated queries and integration
- **Community forum** — Q&A forum for discussion and support directly on the JASPAR website
- **Open access** — all data freely available with no registration required

## Getting Started

1. Visit [JASPAR](https://jaspar.elixir.no/) to search or browse TF binding profiles
2. Use the search function to find profiles by organism, gene name, or protein
3. For programmatic access, use the [RESTful API](https://jaspar.elixir.no/api/) or install pyJASPAR via pip
